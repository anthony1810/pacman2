Installation

Step 1: Opening the project folder location with terminal

Step 2: typing make to compile sourcode

Step 3: type ./pacman.sh <map_name> or ./pacman.sh

Step 4: Enter your name and email as required

Step 5: type ':'' to enter command mode and 'esc' to quit command mode
